Which one should I use?

Use PhPstrap if:

You need users, authentication, modules, database features, admin settings, or a membership system.

Use PhPstrap Lite if:

You just need a fast Bootstrap website, landing page, documentation site, or simple blog without a database.

Tip: Start with Lite for simple sites, then use the full framework when you need users, modules, and app features.
