Build faster with PhPstrap

A lightweight PHP toolkit for membership sites, admin dashboards, modular apps, and static Bootstrap websites.

PhPstrap

Full framework for dynamic PHP apps

Use PhPstrap when your project needs authentication, users, modules, settings, database tables, admin screens, hooks, and extendable app features.

  • Membership sites and dashboards
  • Module system and admin panel
  • Database-backed apps and tools
  • Hooks, settings, auth, and reusable patterns

PhPstrap Lite

Minimal static PHP + Bootstrap starter

Use PhPstrap Lite when you want a clean Bootstrap website with shared PHP includes, reusable sections, SEO meta, blog pages, and no database.

  • No database or installer required
  • Great for landing pages and small business sites
  • Flat-file blog and guide pages
  • Reusable Bootstrap sections and templates

Static Sites

Need something simpler? Use PhPstrap Lite

PhPstrap Lite is the minimal option for static-style PHP websites. It keeps the useful parts — shared includes, Bootstrap sections, SEO meta, and simple pages — without the database, login system, or admin panel.

  • Perfect for landing pages and documentation sites
  • Flat-file blog and guide pages
  • Easy to edit and deploy on shared hosting
