Privacy Policy

This Privacy Policy explains how PhPstrap handles information when you visit this website, view documentation, access project resources, or interact with links to related services.

Summary: PhPstrap is an open-source project website. We aim to collect as little personal information as possible and use basic website data only to improve the project, documentation, and user experience.

1) Who We Are

PhPstrap is a PHP and Bootstrap-based project for building websites, dashboards, membership systems, modules, and static PHP sites through PhPstrap Lite.

The official project website provides documentation, showcase pages, GitHub links, release information, and related project resources.

2) Information We May Collect

We do not require visitors to create an account to browse the public PhPstrap website or documentation.

Depending on how the website is configured, we may collect limited information such as:

  • Basic server logs, such as IP address, browser type, operating system, referring page, and request time.
  • Analytics data, such as page views, popular documentation pages, and general usage trends.
  • Information you voluntarily provide if you contact us, open an issue, submit feedback, or interact with project links.

This information is used to maintain the website, improve documentation, diagnose technical issues, and understand how people use the project.

3) GitHub and External Project Links

PhPstrap links to GitHub repositories, including the main PhPstrap project and PhPstrap Lite. If you visit GitHub, download releases, open issues, star repositories, or submit pull requests, your interaction is governed by GitHub’s own privacy practices.

We are not responsible for the privacy practices of external websites linked from this site.

4) Demo Website

PhPstrap may link to a demo website, such as demo.phpstrap.com.

Demo environments are intended for testing and preview purposes only. Do not submit sensitive, confidential, or production information into a public demo unless the demo page clearly states that it is intended for private use.

5) Cookies

This website may use cookies or local browser storage for basic functionality, preferences, analytics, or embedded third-party services.

Examples may include:

  • Remembering display preferences.
  • Measuring anonymous website traffic.
  • Supporting embedded content or external services.

You can control cookies through your browser settings. Blocking cookies may affect some website functionality.

6) Analytics

We may use privacy-conscious analytics or standard website analytics tools to understand how visitors use the site.

Analytics information may include pages visited, browser type, general location based on IP address, device type, referral source, and approximate visit time.

This information helps us improve documentation, identify broken pages, and prioritize project resources.

7) Server Logs

Like most websites, our hosting provider may automatically collect server logs. These logs may include IP addresses, request URLs, browser user agents, timestamps, and error information.

Server logs are used for security, troubleshooting, abuse prevention, and performance monitoring.

8) How We Use Information

We may use collected information to:

  • Operate and maintain the website.
  • Improve documentation and project resources.
  • Monitor performance and fix technical issues.
  • Protect the website from spam, abuse, or security threats.
  • Respond to inquiries, feedback, or GitHub issues.

We do not sell personal information.

9) Third-Party Services

The PhPstrap website may use or link to third-party services, including but not limited to:

  • GitHub for source code, releases, issues, and collaboration.
  • Bootstrap CDN or other asset CDNs for frontend resources.
  • Font Awesome or icon libraries.
  • Analytics providers, if enabled.
  • Hosting, CDN, DNS, or security providers.

These services may collect information according to their own privacy policies.

10) Data Sharing

We do not sell visitor data. We may share limited information only when necessary to:

  • Operate hosting, analytics, security, or infrastructure services.
  • Comply with legal obligations.
  • Protect the rights, safety, or security of the project, website, or users.
  • Investigate abuse, spam, or attempted attacks.

11) Data Retention

Server logs and analytics data may be retained for a limited period as needed for security, troubleshooting, reporting, and website improvement.

Information submitted through external services, such as GitHub issues or pull requests, may remain available according to those services’ retention policies and public project settings.

12) Security

We use reasonable measures to protect the website and related infrastructure. However, no website or internet transmission is completely secure.

Please avoid submitting passwords, private API keys, confidential files, or sensitive personal information through public demo forms, GitHub issues, or public discussion areas.

13) Children’s Privacy

PhPstrap is a developer-focused project and is not directed at children. We do not knowingly collect personal information from children through this website.

14) Your Choices

You may:

  • Disable cookies in your browser.
  • Use browser privacy settings or tracking protection tools.
  • Avoid submitting personal information through public channels.
  • Contact us if you have questions about privacy or data handling.

15) Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be posted on this page with a revised effective date.

Continued use of the website after changes are posted means you accept the updated policy.
