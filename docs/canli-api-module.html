⚖️ CanLII API Module

The CanLII API Module allows your PhPstrap application to securely store and use a CanLII API key. Once configured, other modules or integrations can access the key to retrieve legal case and legislation metadata directly from the CanLII REST API.

📌 What This Module Does

  • Securely stores your CanLII API key.
  • Provides a simple way for other modules to access the key.
  • Includes a Test Connection tool to verify API access.
  • Supports English (en) and French (fr) API responses.
Note: The CanLII API is read-only and returns JSON metadata about Canadian court decisions and legislation.

🔑 Step 1: Obtain a CanLII API Key

Before using the module, you must request an API key from CanLII.

  1. Visit the CanLII feedback page.
  2. Request developer API access.
  3. Once approved, you will receive an API key.
Important: Keep your API key private. Do not expose it in public repositories.

⚙️ Step 2: Configure the Module

  1. Login to your PhPstrap admin panel.
  2. Navigate to /admin/modules.
  3. Locate the CanLII API module.
  4. Click Settings.

Module Settings

Setting Description
Enabled Enable or disable the CanLII integration.
API Key Your CanLII developer API key.
Language API response language (en or fr).
Test URL The endpoint used to verify your API connection.

🧪 Step 3: Test the API Connection

The module includes a built-in test tool to verify that your API key works correctly.

  1. Open the module test page.
  2. Click Test API Connection.

The module will call the CanLII API using your stored key.

The default test endpoint is:

https://api.canlii.org/v1/caseBrowse/en/

The request sent by the module looks like:

https://api.canlii.org/v1/caseBrowse/en/?api_key=YOUR_KEY

If successful, the API will return JSON listing courts and tribunals.

📚 Example API Response

{
  "caseDatabases": [
    {
      "databaseId": "oncicb",
      "jurisdiction": "on",
      "name": "Criminal Injuries Compensation Board"
    },
    {
      "databaseId": "nlpc",
      "jurisdiction": "nl",
      "name": "Provincial Court of Newfoundland and Labrador"
    }
  ]
}

🧩 Using the Module in Other Integrations

Other modules or services can retrieve the stored API key to perform CanLII API requests. 

$canlii = new CanliiModule();
$apiKey = $canlii->getApiKey();

$url = "https://api.canlii.org/v1/caseBrowse/en/?api_key=" . $apiKey;

$response = file_get_contents($url);
$data = json_decode($response, true);

This allows your application to retrieve:

  • Canadian court decisions
  • Legal citations
  • Legislation metadata
  • Case citation relationships

🛠️ Troubleshooting

  • Connection failed? Verify your API key is correct.
  • Empty results? Ensure the correct language parameter (en or fr) is selected.
  • API errors? Confirm the endpoint supports your request.
Tip: The CanLII API limits responses larger than 10MB. Large results should be requested using pagination parameters like offset and resultCount.

Learn more about the CanLII API at api.canlii.org.
